- ## 简述: 什么是B站美化脚本
  + 🩷 屏蔽视频推送, 避免搜索资料时分心
  + 🩷 去除搜索推荐
  + 🩷 美化, 支持自定义背景

  


  ## Q: 如何打开设置? 如何上传壁纸? 如何管理壁纸?
  A: 点击浏览器右上角油猴或脚本猫图标, 点击 " 设置(点我切换背景图片)"
